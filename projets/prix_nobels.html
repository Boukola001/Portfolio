Analyse des prix nobel : lauréats et contributions

Ce projet vise à étudier les lauréats et les découvertes récompensées par le Prix Nobel, à travers une analyse automatisée de fichiers de données.

Le projet débute par l’ouverture et la lecture du fichier de données au format CSV à l’aide du module Python csv. Cela nous permet de préparer du jeu de données pour les statistiques automatisées : remplissage des valeurs manquantes avec des valeurs par défaut adaptées au contexte, création de colonnes supplémentaires pour optimiser l’analyse, etc...

Les résultats sont automatiquement sauvegardés dans de nouveaux fichiers (csv, xlsx et json) pour faciliter leur récupération et leur présentation.

Nous avons extrait plusieurs statistiques clés du jeu de données :

  • La répartition hommes/femmes parmi les lauréats
  • Le nombre de prix attribués selon la catégorie et le pays
  • L’âge du plus jeune et du plus âgé lors de la rem­ise du prix
Les résultats, synthétisés dans un tableau, apportent une vue d’ensemble sur la diversité, la répartition géographique et les parcours exceptionnels des lauréats.

Analysis of the Nobel Prizes: winners and contributions

This project aims to study the winners and discoveries awarded the Nobel Prize, through automated analysis of data files.

The project begins by opening and reading the data file in CSV format with Python's module csv. This allows us to prepare the dataset for automated statistics: filling missing values ​​with default values ​​adapted to the context, creating additional columns to optimize the analysis, etc.

The results are automatically saved in new files (csv, xlsx and json) to ease their recovery and presentation.

We extracted several key statistics from the dataset:

  • The distribution of men/women among the winners
  • The number of prizes awarded according to category and country
  • The ages of the youngest and oldest at the time of the award ceremony
  • The results, summarized in a table, provide an overview of diversity,
the geographical distribution and exceptional backgrounds of the winners.
